E-Commerce Sales Analysis using MYSQL

Project Overview:
This project demonstrates SQL based analysisof an E-Commerece datset using MYSQL queries.

Data Source: Kaggle E-Commerce Dataset Tables used: Cutomers, orders, products

Objective:
Analyze sales and cutomers data to generate insights such as identifying top selling, finding top customers based on purchases,
analyzing revenue by products categories, monthly sales trends, payment method distribution.

SQL Concepts used:
Select, Where, Order by, Group By, HAving, Inner Join, Aggregate function.
